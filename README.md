# Installation
<p> Step 1 : clone repo and checkout to master branch.</p>
<p> Step 2 : cd server && composer install </p>
<p> Step 3 : php artisan migrate </p> 
<p> Step 4 : php artisan serve </p> 

<p> Step 5 : go to react folder </p> 
<p> Step 6 : npm i </p> 
<p> Step 7 : npm start </p>
